## Getting Started

### Installation and Setup

- Download and Install [Golang](https://golang.org/dl/)

- Install dependencies

```bash
go get ./...
```

### Running

```bash
make run
```

### Testing

```bash
make test
```

### Code Coverage

```bash
make cover
```
